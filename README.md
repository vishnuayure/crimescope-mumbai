## Overview

CrimeScope Mumbai is an end-to-end data analytics project that examines crime patterns, public safety indicators, and law enforcement outcomes in Mumbai using official crime statistics from 2018 to March 2026.

The project leverages data cleaning, SQL-based querying, exploratory data analysis (EDA), statistical techniques, and interactive visualizations to uncover meaningful insights into Mumbai's evolving crime landscape. Through comprehensive analysis and dashboarding, CrimeScope Mumbai aims to transform raw crime data into actionable intelligence for understanding crime trends and public safety challenges.

---

## Key Highlights

- Analyzed over eight years of official Mumbai crime data spanning 2018 to March 2026
- Investigated trends across IPC/BNS crimes, cybercrime, crimes against women, NDPS offences, economic offences, and other major crime categories
- Performed extensive data cleaning, preprocessing, and transformation using Python, Pandas, and NumPy
- Utilized SQL (MySQL) to query, aggregate, and analyze large crime datasets efficiently
- Conducted Exploratory Data Analysis (EDA) to identify long-term trends, anomalies, seasonal patterns, and year-over-year changes in crime incidence
- Applied statistical analysis to evaluate crime distribution and public safety indicators
- Built interactive Power BI dashboards and reports to communicate insights effectively
- Developed visualizations using Matplotlib and Seaborn for detailed trend analysis and storytelling
- Explored predictive modeling techniques using Scikit-learn to forecast crime trends and evaluate future risk patterns

## Project Workflow

**1. Data Collection**
Gathered official Mumbai crime statistics covering 2018 to March 2026.

**2. Data Cleaning and Preparation**
Handled missing values, inconsistencies, and formatting issues. Standardized crime categories and prepared datasets for analysis.

**3. SQL Analysis**
Created queries to aggregate crime data by year, category, and location. Extracted key metrics and trend indicators for further analysis.

**4. Exploratory Data Analysis (EDA)**
Examined crime distributions and historical patterns. Identified spikes, anomalies, and emerging trends.

**5. Statistical Analysis**
Evaluated crime growth rates, category-wise trends, and public safety indicators. Measured changes in crime incidence across different periods.

**6. Predictive Modeling (Optional)**
Engineered features from historical crime records. Built and evaluated forecasting models using Scikit-learn.

**7. Visualization and Reporting**
Created charts and analytical reports using Matplotlib and Seaborn. Developed interactive Power BI dashboards for stakeholder-friendly exploration.

## Getting Started

### Prerequisites

- Python 3.x
- MySQL Server
- Power BI Desktop (for viewing dashboards)
- Jupyter Notebook

### Installation

```bash
# Clone the repository
git clone https://github.com/vishnuayure/CrimeScope-Mumbai.git
cd CrimeScope-Mumbai

# Install required Python packages
pip install -r requirements.txt
```

### Usage

```bash
# Launch Jupyter Notebook to explore the analysis
jupyter notebook notebooks/
```

Open the relevant .pbix file in Power BI Desktop to explore the interactive dashboards.
